Setup Instructions (Short, Clean, Clear)

1. Clone the repository

```bash
git clone git@github.com:Sasha-Mx/Neural.Net-AgenticAI-PROJECT2.git

2. Open the project
```bash
cd Neural.Net-AgenticAI-PROJECT2
```
3. Install dependencies**
```bash
npm install
```
4. Add environment variables**

Create a file named`.env` in the root folder and add the required keys:

```
OPENAI_API_KEY=<your-api-key>
DATABASE_URL=<your-database-url>
MINIO_ACCESS_KEY=<your-minio-key>
MINIO_SECRET_KEY=<your-minio-secret>
```
5. Start the development server**

```bash
npm run dev
```
6. Open the app
